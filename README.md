# VERITAS — Fake News Detector API

> NLP-powered fake news detection using TF-IDF + Logistic Regression, served as a FastAPI REST API with a custom web UI.

## Overview

VERITAS classifies news articles as **REAL** or **FAKE** using:
- **TF-IDF vectorisation** — converts text into weighted keyword features
- **Logistic Regression** — fast, interpretable, strong NLP baseline
- **Keyword explainability** — shows which words drove the prediction
- **FastAPI backend** — async REST API with Swagger docs
- **Custom frontend** — dark editorial UI with animated confidence meter

## Quickstart

```bash
# 1. Clone and install
git clone https://github.com/DrushtiV/fakenews-detector
cd fakenews-detector
pip install -r requirements.txt

# 2. Train the model (sample data built-in)
python model/train.py

# 3. Start the API
uvicorn app.main:app --reload

# 4. Open browser
# UI  →  http://localhost:8000
# API →  http://localhost:8000/docs
```

## Using the Real Kaggle Dataset (Recommended)

1. Download from: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
2. Place `Fake.csv` and `True.csv` in `./data/`
3. Run: `python model/train.py --real-data`

Expected accuracy with real dataset: **~98%**

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/predict` | Classify article text |
| `GET` | `/health` | Health check |
| `GET` | `/examples` | Sample inputs |
| `GET` | `/docs` | Swagger UI |

### Example Request

```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"text": "Scientists confirm 5G towers cause cancer and media is hiding it!"}'
```

### Example Response

```json
{
  "label": "FAKE",
  "confidence": 0.9231,
  "real_probability": 0.0769,
  "fake_probability": 0.9231,
  "top_fake_indicators": ["confirm", "hiding", "cancer", "media"],
  "top_real_indicators": ["scientists", "towers"],
  "word_count": 12,
  "warning": null
}
```
